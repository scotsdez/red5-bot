Red 5 Prototype

A simple prototype to illustrate possible wingman conversational response behaviour to user input.

This is a very simple Javascript chatbot, based on wingman behaviour from X-Wing. Try giving it orders, such as attacking, escorting or ignoring a target, wait for new orders, or evasive manoeuvres. You can view the current understood action, as well as a current world state in JSON format. Try ordering an attack on a friendly target...
